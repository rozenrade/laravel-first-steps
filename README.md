# Laravel First Steps

A simple CRUD (Create, Read, Update, Delete) application built with Laravel — my first project using the framework.

## Prerequisites

Before you begin, ensure you have the following installed on your system:

- **Composer** – Dependency manager for PHP  
  [Download Composer](https://getcomposer.org/download/)

- **PHP** – Included with tools like XAMPP, MAMP, or a local PHP installation  
  [Download XAMPP](https://www.apachefriends.org/)

- **Laravel Installer** – Installed globally via Composer

## Installation

1. **Install Laravel globally**  
   Run the following command in your terminal:

   ```bash
   composer global require laravel/installer
